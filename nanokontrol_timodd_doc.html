nanoKONTROL timod1

K1-K8 depend on knobs-mapping; default: Common Parameters

S1-S8 depend on sliders-mapping; default: Macros

B1-B8 Select cursor-track

B10-B17 Launch single clip

 

K9: set project-tempo

S9: crossfader

B9 = Shift9

B18 = Shift18

 

Play: launch scene and select next scene

Stop: globally return to arrangement and reset automation override

Rec: arm/unarm cursor-track

Rew: select previous scene

FF: select next scene

hold/release Cycle: toggle between alternate tempo and project-tempo (default: half tempo)

hold Cycle + move K9: set alternate tempo. It also affects the project-tempo, until Cycle is released

 

Shift9 +

B1: map knobs to pan

B2: map knobs to send1

B3: map knobs to send2

B4: map knobs to send3

B5: map knobs to parameter (only when sliders are not mapped to parameter)

B6: map knobs to envelope

B7: map knobs to common

B8: map knobs to macro

B10: map sliders to volume

B11: map sliders to send1

B12: map sliders to send2

B13: map sliders to send3

B14: map sliders to parameter (only when knobs are not mapped to parameter)

B15: map sliders to envelope

B16: map sliders to common

B17: map sliders to macro

K9: pre-adjust new tempo (release Shift9 to set new tempo)

S9: - (crossfader disabled)

Play: transport restart

Stop: transport stop

Rec: transport record

Rew: skip back

FF: skip forward

Cycle: set alternate tempo to ¼ speed

 

Shift18 +

B1 – B8: toggle solo

B10: previous preset

B11: next preset

B12: previous preset category

B13: next preset category

B14: -

B15: -

B16: previous parameter page

B17: next parameter page

K9: -

S9: - (crossfader disabled)

Play: transport play

Stop: temporarily mute/unmute master-track

Rec: transport record

Rew: select device to the left

FF: select device to the right

Cycle: set alternate tempo to ½ speed

 

Shift9 + Shift18 +

B1 – B8 -

B10 – B17: track-wise return to arrangement

K9: -

S9: - (crossfader disabled)

Play: activate engine; if active: transport restart

Stop: toggle master-track mute

Rec: transport record

Rew: scroll tracks-page up

FF: scroll tracks-page down

Cycle: toggle loop
